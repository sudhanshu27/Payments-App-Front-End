# Payments-App-Front-End

***Built Front End of a Payments Application:***  

**Include 4 Pages:**  

***1) Sign Up Page***  
***2) Sign In Page***  
***3) Dashboard Page***    
***4) Send Money Page***  

**Step to Run the project locally**  

Install Node Module: ***npm install***  
  
Start project with: ***npm run dev***
